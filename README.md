# mini-hackathon-social-app-WIBBA
A Social App Project Built with JavaScript

## Description
This repository contains a social app project, titled "mini-hackathon-social-app-WIBBA", developed using JavaScript. The project includes a simple web application with an HTML file, a JavaScript file, and a CSS file. The application is hosted live on [Netlify](https://wibba.netlify.app/).

## Features
* **User Interface**: A interactive and user-friendly interface using HTML, CSS, and JavaScript.
* **Real-time Updates**: The web application provides real-time updates, ensuring that users can view the latest information without needing to refresh the page.
* **Social App Functionality**: The application includes social app features, such as user profiles and the ability to share posts, comment, likes & following profiles.
* **Responsive Design**: The web application is designed to be responsive, meaning it can be accessed and used on a variety of devices, including desktops, laptops, tablets, and mobile phones.

## Usage
To use the application, simply open the `index.html` file in a web browser. The application will run without any additional setup. Alternatively, you can view the live application on [Netlify](https://wibba.netlify.app/).

## Project Structure
The project is organized with the following files at the root level:
* `app.js`: The main JavaScript file containing the application logic.
* `index.html`: The main HTML file that serves as the entry point for the application.
* `style.css`: The CSS file that defines the styles for the application.
